# Business Performance Analytics Dashboard

## Overview
This project analyzes retail sales data to evaluate business performance across regions, categories, and time. An interactive Power BI dashboard was built to support data-driven decision-making.

## Objectives
- Track key business KPIs
- Analyze sales trends and seasonality
- Identify top and underperforming regions
- Evaluate category and sub-category profitability

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Data Cleaning & Feature Engineering

## Key KPIs
- Total Sales: $2.30M
- Total Profit: $286K
- Profit Margin: 12.47%
- Total Orders: 5,009
- Total Customers: 793

## Key Insights
- Technology is the most profitable category
- Furniture has low profitability and requires strategic review
- West region contributes the highest sales
- Strong year-end seasonality observed in sales
